历史概述
------------

前身为1152年设立的香山县；1925年，为纪念孙中山而改名为中山。
明朝，香山北部地区开始淤积成陆，此时才与大陆相连。
中山在古代默默无闻，到近代因为其地理、人文、华侨的原因，与海外交流甚多，成为推翻满清建立
共和的桥头堡。

到2010年，中山市户籍人口只有142.3万，祖籍本市的海外华侨和旅居港澳台同胞共80多万人。
虽然土地和人口很少，但因市民思想开明、民企众多，经济总量排在广东第五位，是中国4个不设市辖区的地级市之一。
除了发展经济，中山人还非常注意环境保护，获得过联合国人居奖。

值得注意的是，珠海和澳门曾经都是中山的一部分，现在也是香山文化的继承者。


石岐话
---------

石岐一直中山最重要的地方，现在是市中心的统称。石岐话是粤语的一种，主要流行于广东省中山市
的石岐及南朗镇、珠海市的唐家湾镇、前山一带，与广州粤语可互通，但又不尽相同。

在民国时期石岐话曾盛极一时，原因是孙中山、唐兆仪等名人都是用石岐话来交流。而且随着由
中山人一手创办的四大百货在上海崛起，许多中山人来到当时十里洋场的上海滩工作，于是石岐话在
当时的上海占有一席之位，其中，四大百货公司中的永安公司名字也是用中山拼音，
读作“Wing On”，甚至在中山人开设的食店，上海人和外国人都会用石岐话来点菜。

到20世纪末，石岐话渐渐衰弱，目前仅限于中山和珠海个别区域使用，2010年使用人口约为50万人。


香山文化
---------

香山文化在地缘上是指包括今天的中山、珠海、澳门在内的地域文化。它在本质上集中体现了岭南文化
中粤、闽、客三大民系的文化特征，是中原文化、土著文化、西洋文化、南洋文化相互碰撞和不断融合的产物。


菊花文化
^^^^^^^^^^^

.. image:: my_figs/菊花会.jpg
  :scale: 70 %
  :align: center
  :target: https://zh.wikipedia.org/wiki/小榄菊花会

与日本很相似，菊花是中山人的图腾，每隔60年，逢中国传统干支纪年的甲戌年都将举行一次
盛大的菊花大会，大会期间会每年或数年举行一些小型的菊花会。
1994年的菊花会是离现在最近的一次菊花大会，下一次菊花大会在2054年举行，
因此一个人一生最多只能参加2届菊花大会。

我深深地记得1994年，举办菊花大会时全中山各个镇区每一条街道都摆满了菊花，菊花制作的超大型
艺术品在每一个主干道和空旷地带都能看到，虽然我当时年纪很小，但菊花大会仍然能让我永生难忘。


万人行
^^^^^^^^

.. image:: my_figs/万人行.jpg
  :scale: 120 %
  :align: center
  :target: http://baike.baidu.com/view/1427034.htm

每年年初的慈善万人行活动，是现代中山精神的一种体现。万人行是每年春节
全市市民那天都可以加入到游行队伍中，先到先走，全市直播，队伍中可以看到很多展示中山传统文化的方阵，
也可以看到很多企业、政府单位、社会团体的方阵。
与其它慈善募捐活动不同，万人行是中山全市市民共同参与的活动，是年复一年点点滴滴汇聚成的爱。

近代以来，特别是推翻满清建立共和的阶段，中山人很好地通过行动阐述了中山精神。

小学与中学
=============

我的求学经历一波三折，但我却一直能在最好的学校读书，是十分幸运的。
每个时期都发生了很多故事，我把一些故事整理在《我的经历》里。

中山石岐中心小学
------------------

中山石岐中心小学是城区里最好的小学，我不记得我是怎么进去的了。一年级时考过年级第一，后来
因为多动症等原因，成绩一落千丈，五年级的时候开始认真读书，但到小学毕业，我的成绩在班里
属于中下水平。

在我升初中那一年，中山取消了考试制度！意思就是说过去小学升初中是看成绩的，现在改成电脑
抽签和就近分配。。这对于我这样的学渣来说，是天大的好事。。中山最好的两所公立中学是
中山市第一中学和中山市华侨中学，都在城区里，这两所中学的名额通过全市直播电脑抽签决定，
我那天在小学同学家，和他的家长一起看抽签直播。抽签刚开始5分钟，我就被
抽到了第一中学，收到很多人的祝贺，而我的同学抽到了华侨中学。

最后发现我们班里成绩很好的同学都没有抽到这两所中学，很多人现在都没有联系了，不知道他们
的人生轨迹会变得怎么样。。


三鑫双语学校
----------------

好，问题来了，我最后居然没有去第一中学，而是去了民办的三鑫双语学校！这所民办学校离城里
30分钟车程，要寄宿在学校所以每周只能回家2天，而且这所中学才刚刚成立！很多亲戚朋友都觉得
不可思议。

然而这只有一个目的，就是希望能在高中时考入中山纪念中学，这一所在广东被视为神一样的学校。

小学时期¶

一年级到四年级时，学校的硬件条件比较差，在旧城区里，旁边是一个菜市场，交通也不方便， 只有一条小路可以出入。五年级时，我们搬到了东区，学校设施有了翻天覆地的变化。

旧校址的硬件设施¶

我刚才提到旧校址的硬件条件差，我这里可以介绍一下， 当时的学校外墙是黄色油漆的，显得很脏。我们每天那饭盒到学校吃饭，饭堂会把一大桶饭拿到课室里分， 大家吃完后，就到把剩饭倒到外面的白色大塑料桶里。大家要知道，那种白色塑料桶对小学生来说是很高的， 有时候不小心把调羹仍到桶里了，想要捡是很困难的，而且桶里还很臭，有一次我的调羹掉到桶里了，我尝试 了好几次才捡出来，然后洗了很久很久。

一年级到四年级我感觉过得很慢，但很多事情却又很模糊，我记得当时是每升一年级，就搬一层楼。则一年级 在一楼，二年级在二楼，以此类推。

年级第一¶

一年级时，我们只学语数英3科，我阴差阳错地都考了满分，结果发成绩单那天老师给了一个年级第一名的 证书，那天我爸接我，我们开心得不的了。

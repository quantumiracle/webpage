广东人是谁？¶

广东的土地上主要生活着广府人、潮汕人和客家人。但这里指但广东人是特指广府人，即说粤语的人。 广府文化是指以广州为核心、以珠江三角洲为通行范围的粤语文化它从属于岭南文化，在岭南文化中影响最大。 中山属于 然而广府人主要由早期移民与古越族杂处同化而成。

大粤民国，粤语拉丁化

从历史上来讲

广东独立，简称“广独”或“粤独”，某些“广东自治”的言论也被指为“广东独立”，是存在于网络上的一种思潮， 部分网民提出广东从中国独立出来，以发展“独立”或“自治”。其言论主要倡导广东省的经济要从中国所独立 出来，即所谓“广东自治”，这也可以理解为民初时期的联省自治这一思想的延续，类似美国的联邦模式。 而更为激进的部分人甚至倡导行政独立。原因是有些人认为广东古时为百越之地，中原入侵才至并入中华版图， 而更主要的原因是有广东人不满现中国政府对广东的不公政策，包括语言打压、言论控制以至本地文化北方化等。

您好，我叫 `董豪`_ ，这里是我的私人数据库，能在茫茫大海中在此相遇真的很难得！
我把这里当作是一个永久的私人数据库，记录一些我的经历、不同时期的思想以及我做过的一些事情。
其实我不太希望被人发现这个数据库，我的想法是:只要互联网存在，这些数据将被永久保存，
以供后人回顾20世纪末到21世纪末的这段历史。

这里的内容
----------

这个数据库包含如下内容。

* **关于我**: 您或许不会在乎我是谁，但了解我的基本背景可以让您更好地了解这段历史。

* **我的经历**: 很高兴能活在中国大陆高速发展的时期，也有幸有一段英国求学的经历。

* **我的思想**: 在不同时期我对不同事物的看法很不一样。

* **我的项目**: 人生是由大大小小的项目组成的，作为一个理工科毕业的我有很多工程和科学项目，
  但我也很喜欢做其它有意思的事情。
